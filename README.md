 ### 레포 만들기
> 레포생성 하고 README.md 추가부터 전부 다 입력

 ### 브랜치
> 브랜치 생성
> git branch "브랜치명" 
> ex) git branch MinJun

> 브랜치 변경 
>  git checkout "브랜치명"
>  git checkout Minjun
<br/>
